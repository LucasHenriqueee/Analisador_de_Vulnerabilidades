# Códigos Python

Nessa pasta encontra-se os códigos pythons usados para poder criar o cenário no GNS3.

## Descição dos arquivos

- O arquivo [Criador_TopologiaGNS3_4.0.py](Criador_TopologiaGNS3_4.0.py) contêm as funções métodos para criar os a topologia e gerar no arquivo Cenário.gns3.
- O arquivo [checar_SCL_2.py](checar_SCL_2.py) é um arquivo usado para extrair informações do documento e SCL e mandar as informações para o script Criador_TopologiaGNS3_4.0.py.

Para que funcione é necessário verificar o diretório do arquvio .gns3 onde foi salvo e apontar-lo no script. Além disso, verificar a versão do GNS3 e as imagens dockerfile necessárias para elaboração dos dispostivos dentro do GNS3.
